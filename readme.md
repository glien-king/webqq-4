功能需求
====
* 用户可以与好友一对一聊天，消息实时收发<br/>
* 可以显示好友在线状态<br/>
* 可以群聊，有群列表和成员列表<br/>
* 有新消息提示、最近对话列表<br/>
* 有好友列表，可以搜索、添加某人为好友<br/>
* 用户可以搜索和添加群<br/>
* 每个群有管理员可以审批用户的加群请求，群管理员可以用多个，群管理员可以删除、添加、禁言群友<br/>
* 可以与聊天室里的人进行临时会话(与qq群一样)<br/>
* 可以在群中发图片<br/>
* 可以与好友一对一发文件<br/>
